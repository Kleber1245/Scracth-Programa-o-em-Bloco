O que são os Blocos de Operadores?

No Scratch, o botão "Operators" (ou "Operadores") é utilizado para realizar operações matemáticas, comparações lógicas e manipulação de texto nos seus projetos.

Principais Blocos de Operadores:

  • [ ] + [ ]: Realiza a adição de dois números.
  • [ ] - [ ]: Realiza a subtração de dois números.
  • [ ] * [ ]: Realiza a multiplicação de dois números.
  • [ ] / [ ]: Realiza a divisão de dois números.
  • Número aleatório entre [ ] e [ ]: Gera um número aleatório dentro de um intervalo especificado.
  • [ ] > [ ]: Verifica se o primeiro valor é maior que o segundo.
  • [ ] < [ ]: Verifica se o primeiro valor é menor que o segundo.
  • [ ] = [ ]: Verifica se o primeiro valor é igual ao segundo.
  • [ ] e [ ]: Realiza a operação lógica "E" entre duas condições.
  • [ ] ou [ ]: Realiza a operação lógica "OU" entre duas condições.
  • não [ ]: Realiza a operação lógica "NÃO" em uma condição.
  • juntar [ ] com [ ]: Concatena duas strings de texto.
  • letra [ ] de [ ]: Retorna a letra em uma posição específica de uma string.
  • comprimento de [ ]: Retorna o número de caracteres em uma string.
  • [ ] contém [ ]?: Verifica se uma string contém outra string.
  • resto de [ ] / [ ]: Retorna o resto da divisão de dois números.
  • arredondar [ ]: Arredonda um número para o inteiro mais próximo.
  • absoluto de [ ]: Retorna o valor absoluto de um número.
  • piso de [ ]: Retorna o maior inteiro menor ou igual a um número.
  • teto de [ ]: Retorna o menor inteiro maior ou igual a um número.
  • raiz quadrada de [ ]: Calcula a raiz quadrada de um número.
  • seno de [ ]: Calcula o seno de um ângulo (em graus).
  • cosseno de [ ]: Calcula o cosseno de um ângulo (em graus).
  • tangente de [ ]: Calcula a tangente de um ângulo (em graus).
  • arco seno de [ ]: Calcula o arco seno de um número (em graus).
  • arco cosseno de [ ]: Calcula o arco cosseno de um número (em graus).
  • arco tangente de [ ]: Calcula o arco tangente de um número (em graus).
  • ln de [ ]: Calcula o logaritmo natural de um número.
  • log de [ ]: Calcula o logaritmo base 10 de um número.
  • e ^ [ ]: Calcula a exponencial de um número (e elevado à potência do número).
  • 10 ^ [ ]: Calcula 10 elevado à potência do número.

Os blocos de operadores são fundamentais para adicionar lógica e dinamismo aos seus projetos, permitindo realizar cálculos, tomar decisões e manipular dados de texto.
