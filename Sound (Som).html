O que são os Blocos de Som?

No Scratch, o botão "Sound" (ou "Som") é utilizado para adicionar e controlar sons e músicas nos seus projetos, tornando-os mais interativos e expressivos.

Principais Blocos de Som:

  • Tocar som [ ] até o fim: Reproduz um som selecionado e espera até que ele termine de tocar para continuar o script.
  • Tocar som [ ]: Inicia a reprodução de um som selecionado imediatamente, sem esperar que termine para continuar o script.
  • Parar todos os sons: Interrompe a reprodução de todos os sons que estão tocando no momento.
  • Mudar efeito de som [ ] para [ ]: Altera efeitos de som como volume, panorâmica (esquerda/direita), etc.
  • Limpar efeitos sonoros: Remove todos os efeitos sonoros aplicados.
  • Mudar volume para [ ] %: Define o volume geral dos sons para uma porcentagem específica.
  • Aumentar volume em [ ]: Aumenta o volume atual dos sons em um valor especificado.
  • Diminuir volume em [ ]: Diminui o volume atual dos sons em um valor especificado.
  • Definir efeito de [ ] para [ ]: Define um efeito sonoro específico com um valor.
  • Mudar efeito de [ ] por [ ]: Altera o valor de um efeito sonoro específico.

Com os blocos de som, você pode adicionar música de fundo, efeitos sonoros para eventos e até mesmo fazer seus sprites falarem, enriquecendo a experiência dos seus projetos.
