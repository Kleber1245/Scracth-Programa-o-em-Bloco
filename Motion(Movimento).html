O que são os Blocos de Movimento?

No Scratch, o botão "Motion" (ou "Movimento") é utilizado para controlar como os sprites se movem e giram na tela, definindo sua posição e orientação.

Principais Blocos de Movimento:

  • Mover [ ] passos: Move o sprite para frente no sentido de sua direção atual.
  • Girar [ ] graus para a direita: Gira o sprite no sentido horário em um ângulo especificado.
  • Girar [ ] graus para a esquerda: Gira o sprite no sentido anti-horário em um ângulo especificado.
  • Apontar na direção [ ]: Define a direção para a qual o sprite está voltado.
  • Apontar para [ ]: Faz o sprite se voltar para outro sprite ou para o ponteiro do mouse.
  • Ir para x: [ ] y: [ ]: Define a posição exata do sprite nas coordenadas X e Y.
  • Mudar x por [ ]: Altera a posição horizontal do sprite em um valor especificado.
  • Definir x para [ ]: Define a posição horizontal do sprite para um valor específico.
  • Mudar y por [ ]: Altera a posição vertical do sprite em um valor especificado.
  • Definir y para [ ]: Define a posição vertical do sprite para um valor específico.
  • Mudar [ ] por [ ]: Altera a posição X ou Y do sprite gradualmente.
  • Se tocar na borda, rebater: Faz o sprite mudar de direção ao tocar na borda da tela.
  • Definir estilo de rotação para [ ]: Define como o sprite deve rotacionar (normal, apenas face esquerda-direita, ou não rotacionar).
  • Posição x: Retorna a coordenada X atual do sprite.
  • Posição y: Retorna a coordenada Y atual do sprite.
  • Direção: Retorna a direção atual do sprite (em graus).

Os blocos de movimento são essenciais para animar seus personagens, fazer objetos interagirem e controlar a dinâmica visual dos seus projetos no Scratch.
