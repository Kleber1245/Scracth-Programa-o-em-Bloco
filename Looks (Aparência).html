O que são os Blocos de Aparência?

No Scratch, o botão "Looks" (ou "Aparência") é utilizado para controlar a forma como os sprites e o palco se apresentam visualmente no seu projeto.

Principais Blocos de Aparência:

  • Dizer [ ] por [ ] segundos: Mostra uma fala do sprite na tela por um tempo determinado.
  • Dizer [ ]: Mostra uma fala do sprite na tela até ser alterado.
  • Pensar [ ] por [ ] segundos: Mostra um balão de pensamento do sprite por um tempo determinado.
  • Pensar [ ]: Mostra um balão de pensamento do sprite até ser alterado.
  • Mudar fantasia para [ ]: Altera a imagem do sprite para uma fantasia específica.
  • Próxima fantasia: Muda para a próxima fantasia na lista de fantasias do sprite.
  • Mudar cenário para [ ]: Altera o fundo do palco para um cenário específico.
  • Próximo cenário: Muda para o próximo cenário na lista de cenários do palco.
  • Mudar tamanho para [ ] %: Define o tamanho do sprite para uma porcentagem específica do seu tamanho original.
  • Mudar efeito [ ] por [ ]: Altera efeitos visuais como cor, brilho, transparência, etc.
  • Limpar efeitos gráficos: Remove todos os efeitos visuais aplicados ao sprite ou palco.
  • Mostrar: Torna o sprite visível na tela.
  • Esconder: Torna o sprite invisível na tela.
  • Ir para a camada [ ]: Define a ordem de sobreposição do sprite em relação a outros sprites (frente ou trás).

Com esses blocos, você pode dar vida aos seus personagens e cenários, criando animações, reações visuais e tornando seus projetos muito mais expressivos.
